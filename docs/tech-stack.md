# Tecnologias Utilizadas

## Apps Script
- Funções para manipulação de planilhas
- Geração de documentos em PDF
- APIs internas do Google

## HTML + CSS + JavaScript
Interface Front-End leve que roda dentro do WebApp.

## Google Sheets
Usado como banco de dados para:
- Materiais
- OS cadastradas
- Documentos gerados

## Design Técnico
- Comunicação assíncrona entre front-end e Apps Script usando `google.script.run`.
- Separação clara entre lógica de dados e interface.
